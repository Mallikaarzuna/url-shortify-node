users API

Routes Operation
(HTML Response)

- GET / Home page (Simple html title)
- GET /users List of users name in html

(API Response)

- GET /api/users Users JSON Data
- GET /api/users/1 Specific User ID data
- POST /api/users Create a new user
- PUT /api/users/2 Update specific user id data
- DELETE /api/users/1 Delete a specific user
